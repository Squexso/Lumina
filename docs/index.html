Install in 3 steps

  1. Click Download above, then right-click the ZIP → Extract All
  2. Open the folder → double-click LuminaMC.exe
  3. First time only: if Windows warns → More info → Run anyway

It's safe — the warning just means the app isn't code-signed (a paid certificate). A desktop shortcut is created automatically on first launch.
